## 📂 Project Structure

```
src/
├── app/
│   ├── layout.tsx       # SEO metadata, JSON-LD schema
│   ├── page.tsx         # Main page
│   ├── sitemap.ts       # Dynamic sitemap
│   └── robots.ts        # Dynamic robots.txt
├── components/
│   ├── sections/        # Hero, About, Experience, Projects...
│   ├── layout/          # Navbar, LenisProvider
│   └── ui/              # Cursor, Chatbot, animations
public/
├── og-image.png         # Social media preview image
├── robots.txt           # Static robots fallback

└── video.mp4            # Hero background video
```
